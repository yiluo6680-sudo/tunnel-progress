# 隧道工程施工进度及计量可视化系统

将隧道桩号区间映射为线性可视化进度条，一眼看清施工进度和计量状态。

## 快速启动

```bash
cd tunnel-progress
source .venv/bin/activate
streamlit run app.py
```

浏览器打开 `http://localhost:8501`，点击左侧 **"📥 加载示例数据"** 立即体验。

## 功能

- **线性进度图**：左线/右线并行显示，按桩号比例绘制
- **多图层**：开挖 / 初支 / 二衬 / 仰拱 / 计量状态分层显示
- **状态颜色**：灰色=未施工、蓝色=已完工未计量、橙色=申报中、绿色=已计量、**黄色⭐=可计量**
- **KPI面板**：总进度、已计量总额、可计量（待申报）总额
- **可计量区段高亮**：自动识别二衬已完成但未计量的区段
- **过滤器**：按线别、围岩级别、工序、状态筛选
- **报表导出**：可计量区段明细导出 CSV

## 数据格式

上传三张 Excel/CSV 表（内置示例数据可直接体验）：

### 表1：设计主轴线
必需列：隧道名称、线别、起始桩号、终点桩号、围岩级别

### 表2：施工进度台账
必需列：线别、起始桩号、终点桩号、工序名称（开挖/初支/二衬/仰拱）、完成日期

### 表3：计量支付台账
必需列：线别、起始桩号、终点桩号、计量状态（已审批/申报中/未计量）、已计量金额

桩号格式示例：`K85+147` 或 `ZK85+147`
